Three Ways To Add CSS in HTML

  1. 1 - Inline CSS - <div style="background-color: yellow;"></div>
  2. 2 - Internal CSS - in this Way we add style tag in HTML head and give style to the selectors like for example: <style> div{ background-color: aqua; display: flex; } </style>
  3. 3 External CSS - for this We create a External CSS file Which Is Linked with Our HTML Page Using Link Tag <link rel="stylesheet" href="style.css">
